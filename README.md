# Nature-Walk (Unity)
A calm relaxing walk in the jungle, to calm your mind and explore the peaceful nature sight. The scene is made with Unity and lets you walk and explore the scene and enjoy it.

### How to use
To start your journey you can easily clone the repository or manually download the files and then run the project.
For the best experience and prevent facing any error try using the latest version of Unity because it was made with Unity6.
```
git clone https://github.com/bardiaAsraary/Nature-Walk-Unity-/tree/main
```
